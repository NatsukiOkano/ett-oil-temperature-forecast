# 変圧器オイル温度予測 PoC（ETT Oil Temperature Forecast）

ETTデータセット（ETTh1）を用いて、変圧器のオイル温度（OT）の1時間先予測を行うPoC（技術検証）です。線形回帰・SimpleLSTM（1層LSTM）・UNetResidualLSTMBlock（U-Net構造＋残差接続）の3手法を比較検証し、予防保全への活用可能性を検証しました。

本リポジトリは、株式会社Athena Technologies MLエンジニア選考課題の提出物です。

## 背景・目的

変圧器のオイル温度は、設備劣化・事故リスクと密接に関わる重要な監視指標です。従来は経験則に基づく閾値監視が中心でしたが、事後対応型の保全から予測に基づく予防保全への移行が求められています。本PoCでは、公開ベンチマークであるETTデータセットを用いてOTの将来予測モデルを構築し、実際の運用・保全判断にどの程度価値を出せそうかを検証しました。

## 使用データ

- **ETT（Electricity Transformer Temperature）データセット**（`ETTh1.csv`）
- 出典：[zhouhaoyi/ETDataset](https://github.com/zhouhaoyi/ETDataset)
- 1時間単位で、HUFL・HULL・MUFL・MULL・LUFL・LULL（各種負荷データ）とOT（オイル温度）を記録

## アプローチ

EDAの結果、OTは直前1時間の値（OT_lag1）に強く依存し、短期的に自己回帰的な変動を示すことが分かりました。この知見を踏まえ、シンプルな手法から複雑な手法まで、段階的に精度を比較する設計としました。

| # | モデル | 概要 |
|---|---|---|
| 1 | 線形回帰 | OT_lag1・hour・HUFL移動平均などの特徴量を用いたベースライン |
| 2 | SimpleLSTM | PyTorch標準の `nn.LSTM` による1層構成の時系列モデル |
| 3 | UNetResidualLSTMBlock | U-Net構造（時系列の圧縮・復元・スキップ接続）と残差接続を組み合わせた独自実装のカスタムLSTMアーキテクチャ |

すべてのモデルについて、目的変数（OT）を標準化した上で学習し、勾配クリッピング・学習率スケジューラ（ReduceLROnPlateau）・Early Stoppingを導入して学習を安定化させました。最終評価では、標準化前の単位（℃）に逆変換した上でMSE/RMSEを算出し、3モデルを公平に比較しています。

## 結果

| Model | MSE | RMSE (℃) |
|---|---|---|
| **線形回帰** | **0.4179** | **0.6465** |
| SimpleLSTM | 0.7165 | 0.8465 |
| UNetResidualLSTMBlock | 0.7258 | 0.8519 |

複雑な深層学習モデルよりも、シンプルな線形回帰の方が高精度という結果になりました。これは、OTの変動が直前の値に強く規定される自己回帰的な性質を持つため、線形回帰がこの関係を直接的かつ最適に利用できた一方、LSTM系はゲート機構を通じて間接的にこの関係を学習する必要があり、学習データ規模（約14,000サンプル）の中ではその複雑さを十分に活かせなかった可能性があると考えています。

## 現時点の限界・今後の改善方針

- 予測対象は「1時間先」のみであり、複数時刻先（多段階）の予測については未検証です。
- `window_size`・`hidden_dim` などのハイパーパラメータについては、網羅的な探索を行っておりません。
- 学習データ規模は、深層学習モデルの強みを十分に発揮するには限定的である可能性があります。
- 今後は、多段階予測への拡張、ハイパーパラメータ探索、データを拡充した上での再検証を行う予定です。
